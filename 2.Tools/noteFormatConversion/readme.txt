这是一个辅助你转变音符格式的程序，可以将较容易输入的字符串转变为C语言可接受的数组。
需要准备的输入：
在一个文件夹下的三个文件：octave_x.in与rhythm_x.in与tune_x.in。其中x是该歌曲的序号。
在执行该程序的过程中，需要提供两个参数：存放这三个.in文件的文件夹地址（注意不要以反斜杠\结尾）；x的值。

接下来，建议先查看example文件夹下的三个.in文件，再继续阅读。
输入文件的文件需要具备以下性质：
octave_x.in中，只包括数字、负号。其中，带符号的数字表示这个音的相对高低。例如，如果定义C大调下，440Hz为DO的八度为0，那么以880Hz为DO的八度就定义为1。
rhythm_x.in中，只包括1, 2, 4, 8, s这几个时值，支持附点符号。其中1表示全音符，2表示二分音符，...，s表示16分音符。本程序暂不支持输出三连音、六连音等时值。
tune_x.in中，只包括0-7这几个数字。

最后，在计算机上安装python后，输入：
python "xxx\format_conversion\main.py"
即可调用该程序。其中xxx是format_conversion的上级目录地址。

本程序的正确使用方法并不是一味迎合本程序的要求，而是应该在阅读理解该程序的基础上，对相应的内容作出符合自己设计框架、符合自己理解的修改。例如，对于了解乐理的同学，可以将tune.py中的DO, RE, MI, FA, SOL, LA, SI修改成为C, D, E, F, G, A, B。
